Zaloguj się kontem Microsoft 365, aby uzyskać dostęp do danych na SharePoint.

⚠️ Wymagana konfiguracja — uzupełnij CONFIG na górze pliku HTML

  1. Zarejestruj aplikację w Azure Portal → App registrations → New registration
    Redirect URI: Single-page application (SPA) → wklej URL tej strony
  2. Skopiuj Application (client) ID i Directory (tenant) ID do CONFIG
  3. W Azure → API permissions dodaj: Sites.ReadWrite.All, User.Read (Microsoft Graph, Delegated)
  4. Na SharePoint utwórz listę o nazwie ListaObecnosci z kolumnami:
    Title (text) · Miesiac (text) · Rok (number) · Statusy (Multiple lines of text)
  5. Zapisz plik HTML na SharePoint/OneDrive i udostępnij użytkownikom
